Guide Complet - NumPy, Pandas, Matplotlib & Seaborn

Documentation technique pour maîtriser les calculs numériques et la visualisation de données

NumPy - Calcul numérique & Tableaux

À propos de NumPy : NumPy est utilisé pour les calculs mathématiques rapides sur des blocs de données du même type.
Conseil NumPy est 100x plus rapide qu'une boucle Python classique. Utilisez-le pour toutes vos opérations mathématiques.
Attention Les indices NumPy commencent à 0 (pas à 1). np.arange(0, 10) produit [0,1,2...9] (pas 10).
Débutant Fréquent

1. Création de Tableaux (Arrays)

np.array([1, 2, 3])
→ Crée un tableau à partir d'une liste
np.zeros(10)
→ Crée un tableau rempli de zéros (10 éléments)
np.ones((3, 3))
→ Crée une matrice 3x3 remplie de uns
np.arange(0, 10, 2)
→ Crée une suite de nombres de 0 à 9 par pas de 2
→ Résultat : [0, 2, 4, 6, 8]
Débutant Essentiel

2. Statistiques de Base

np.mean(array)
→ Calcule la moyenne des éléments du tableau
np.max(array)
→ Retourne la valeur maximale
np.min(array)
→ Retourne la valeur minimale
np.sum(array)
→ Calcule la somme de tous les éléments
np.std(array)
→ Calcule l'écart-type
np.ptp(array)
→ Calcule l'amplitude thermique (Peak-To-Peak = Max - Min)
Important Avancé

3. Manipulation & Filtrage

array[array > 10]
→ Isole les valeurs supérieures à 10
np.sum(array < 0)
→ Compte le nombre de valeurs négatives
np.round(array, 2)
→ Arrondit les valeurs à 2 décimales

Pandas - Manipulation de Tableaux/Données

À propos de Pandas : Pandas est parfait pour structurer, nettoyer et analyser des données tabulaires (comme Excel).
Bonne pratique Toujours commencer par df.info() et df.describe() pour comprendre vos données avant de les analyser.
Attention Les modifications sur un DataFrame ne s'appliquent pas in-place par défaut. Utilisez inplace=True ou réassignez : df = df.dropna()
Débutant

1. Création et Chargement

pd.DataFrame(data)
→ Crée un DataFrame à partir d'un dictionnaire
df = pd.read_csv('fichier.csv')
→ Lit un fichier CSV
df = pd.read_excel('fichier.xlsx')
→ Lit un fichier Excel
Débutant Essentiel

2. Inspection des Données

df.head(n)
→ Affiche les n premières lignes du DataFrame
df.info()
→ Affiche les informations du tableau (types de colonnes, valeurs manquantes, etc.)
df.describe()
→ Affiche les statistiques rapides (moyenne, min, max de toutes les colonnes numériques)
Débutant Très fréquent

3. Sélection & Filtrage

df['Nom_Colonne']
→ Sélectionne une colonne spécifique
df[df['Vente'] > 1500]
→ Filtre les lignes où la colonne 'Vente' est supérieure à 1500
df.loc[df['Benefice'].idxmax()]
→ Retourne la ligne qui contient le bénéfice maximal
df.loc[ligne, colonne]
→ Accès à une cellule spécifique par ligne et colonne
Important Avancé

4. Calculs & Groupements

df['Total'] = df['A'] + df['B']
→ Crée une colonne calculée 'Total' (somme de A et B)
df['Vente'].sum()
→ Calcule la somme de la colonne 'Vente'
df['Vente'].mean()
→ Calcule la moyenne de la colonne 'Vente'
df.groupby('Categorie')['Vente'].sum()
→ Groupe par catégorie et calcule la somme des ventes par groupe
df.sort_values(by='Vente', ascending=False)
→ Trie les données par la colonne 'Vente' en ordre décroissant
Critique Compétence clé

5. Nettoyage (Data Cleaning)

df.dropna()
→ Supprime les lignes contenant des valeurs vides (NaN)
df.fillna(0)
→ Remplace les valeurs vides par 0
pd.to_datetime(df['Date'])
→ Convertit la colonne 'Date' en format datetime
Avancé Puissant

6. Syntaxes Avancées : Analyse & Nettoyage

df.isna() / df.isnull()
→ Retourne un booléen pour chaque valeur manquante (NaN)
df.isnull().sum()
→ Compte le nombre de valeurs manquantes par colonne
df.dropna(subset=['Colonne1', 'Colonne2'])
→ Supprime les lignes où certaines colonnes sont NaN
df.fillna(method='ffill')
→ Remplit les NaN avec la valeur précédente (forward fill)
df['Colonne'].duplicated().sum()
→ Compte le nombre de doublons dans une colonne
df.drop_duplicates(subset=['Colonne'], keep='first')
→ Supprime les doublons en gardant la première occurrence
df['Colonne'].str.replace('ancien', 'nouveau')
→ Remplace du texte dans les valeurs de colonne (texte)
df['Colonne'].str.upper() / df['Colonne'].str.lower()
→ Convertit le texte en majuscules/minuscules
df['Colonne'].str.strip()
→ Supprime les espaces au début et fin du texte
df['Colonne'].astype(int) / df['Colonne'].astype(str)
→ Convertit le type de données (int, float, str, etc.)
df.apply(lambda x: x * 2)
→ Applique une fonction personnalisée à chaque ligne/colonne
df.groupby(['Cat1', 'Cat2'])['Vente'].agg(['sum', 'mean', 'count'])
→ Agrégation multiple par groupes (plusieurs fonctions)
df.pivot_table(values='Vente', index='Mois', columns='Region', aggfunc='sum')
→ Crée un tableau croisé dynamique (pivot)
df.merge(df2, on='Clé', how='left')
→ Fusionne deux DataFrames (inner, left, right, outer)
df[df['Colonne'].isin(['A', 'B', 'C'])]
→ Filtre les lignes où Colonne est dans une liste
df.query('Age > 30 and Ville == \"Paris\"')
→ Filtre avec syntaxe SQL-like (plus lisible)
df.rename(columns={'Ancien': 'Nouveau'})
→ Renomme les colonnes du DataFrame
df.drop(['Colonne1', 'Colonne2'], axis=1)
→ Supprime des colonnes (axis=1 pour colonnes, 0 pour lignes)
df.reset_index(drop=True)
→ Réinitialise l'index après filtrage/suppression
df.sort_values(by=['Col1', 'Col2'], ascending=[True, False])
→ Trie par plusieurs colonnes avec ordres différents
df['Colonne'].value_counts()
→ Compte les occurrences uniques (fréquence)
df.corr()
→ Calcule la matrice de corrélation entre colonnes numériques
pd.cut(df['Colonne'], bins=5)
→ Divise les données en intervalles (bins) pour discrétisation
df.to_csv('fichier.csv', index=False)
→ Exporte le DataFrame en CSV (json, excel, sql, etc.)

Matplotlib - Visualisation des Données

À propos de Matplotlib : Matplotlib est la bibliothèque de base pour créer des graphiques, des histogrammes et des visualisations statiques en Python.
Conseil Pour des graphiques plus beaux et modernes, utilisez Seaborn qui est construit sur Matplotlib mais offre des styles par défaut supérieurs.
Débutant Essentiels

1. Création de Figures et Subplots

import matplotlib.pyplot as plt
→ Importe la bibliothèque Matplotlib
plt.figure(figsize=(10, 6))
→ Crée une figure avec taille 10x6 pouces
fig, ax = plt.subplots(2, 2)
→ Crée une grille 2x2 de subplots (4 graphiques)
plt.subplot(2, 3, 1)
→ Sélectionne le subplot 1 dans une grille 2x3
Débutant Très fréquent

2. Tracer des Graphiques (Plots)

plt.plot(x, y)
→ Trace une courbe linéaire
plt.scatter(x, y)
→ Trace un nuage de points (scatter plot)
plt.bar(categories, values)
→ Crée un diagramme en barres
plt.hist(data, bins=20)
→ Crée un histogramme avec 20 barres
plt.pie(values, labels=labels)
→ Crée un diagramme circulaire (pie chart)
plt.boxplot(data)
→ Crée un diagramme en boîte (box plot)
Important

3. Personnalisation des Graphiques

plt.title('Titre du graphique')
→ Ajoute un titre au graphique
plt.xlabel('Axe X') / plt.ylabel('Axe Y')
→ Ajoute les étiquettes des axes X et Y
plt.legend()
→ Affiche la légende du graphique
plt.grid(True)
→ Ajoute une grille de fond
plt.xlim(0, 100) / plt.ylim(0, 50)
→ Définit les limites des axes X et Y
plt.xticks(rotation=45)
→ Fait pivoter les étiquettes de l'axe X de 45°
Important

4. Style et Couleurs

plt.plot(x, y, color='red', linewidth=2)
→ Trace une ligne rouge d'épaisseur 2
plt.scatter(x, y, s=100, alpha=0.5)
→ Trace des points avec transparence (alpha=0.5)
plt.style.use('seaborn')
→ Applique le style 'seaborn' à tous les graphiques
plt.rcParams['figure.figsize'] = (12, 8)
→ Définit la taille par défaut des figures
Débutant

5. Affichage et Sauvegarde

plt.show()
→ Affiche le graphique dans une fenêtre
plt.savefig('graphique.png')
→ Sauvegarde le graphique en PNG
plt.savefig('graphique.pdf', dpi=300)
→ Sauvegarde en PDF avec haute résolution (300 dpi)
plt.tight_layout()
→ Ajuste automatiquement l'espacement des subplots
plt.close()
→ Ferme la figure actuelle (libère la mémoire)
Avancé

6. Syntaxes Avancées et Utiles

plt.text(x, y, 'Texte')
→ Ajoute du texte à une position (x, y) du graphique
plt.annotate('Label', xy=(x, y), xytext=(x+10, y+10))
→ Ajoute une annotation avec flèche pointant vers (x, y)
ax.axhline(y=10, color='red', linestyle='--')
→ Ajoute une ligne horizontale à y=10 (pointillée rouge)
ax.axvline(x=5, color='blue', linestyle=':')
→ Ajoute une ligne verticale à x=5 (pointée bleu)
plt.plot(x, y, marker='o', linestyle='-', linewidth=2)
→ Personnalise marqueurs, style et épaisseur de ligne
ax.fill_between(x, y1, y2, alpha=0.3, color='cyan')
→ Remplit l'area entre deux courbes avec transparence
plt.colorbar(im)
→ Ajoute une barre de couleur pour les images/heatmap
ax.set_xscale('log')
→ Utilise une échelle logarithmique pour l'axe X (ou 'linear')
fig.suptitle('Titre global')
→ Ajoute un super-titre pour toute la figure
ax.invert_xaxis() / ax.invert_yaxis()
→ Inverse l'ordre des valeurs sur l'axe X ou Y

Seaborn - Visualisation Statistique

À propos de Seaborn : Seaborn est construite sur Matplotlib et offre des graphiques statistiques plus complexes et esthétiques. Elle s'intègre parfaitement avec Pandas.
Bonne pratique Combinez Seaborn avec Pandas DataFrames pour une workflow optimale : sns.scatterplot(data=df, x='col1', y='col2')
Débutant

1. Chargement et Configuration

import seaborn as sns
→ Importe la bibliothèque Seaborn
sns.set_theme(style='darkgrid')
→ Définit le style global ('darkgrid', 'whitegrid', 'dark', 'white')
sns.set_palette('husl')
→ Définit la palette de couleurs
sns.color_palette()
→ Affiche la palette de couleurs actuelle
Débutant Très utile

2. Graphiques Statistiques

sns.scatterplot(data=df, x='X', y='Y')
→ Crée un nuage de points avec options statistiques
sns.lineplot(data=df, x='X', y='Y')
→ Trace une courbe linéaire avec intervalle de confiance
sns.barplot(data=df, x='Categorie', y='Vente')
→ Crée un diagramme en barres avec moyennes estimées
sns.countplot(data=df, x='Categorie')
→ Compte et affiche la fréquence de chaque catégorie
sns.boxplot(data=df, x='Categorie', y='Valeur')
→ Crée des diagrammes en boîte pour comparer distributions
sns.violinplot(data=df, x='Categorie', y='Valeur')
→ Crée un graphique en violon (distribution détaillée)
Important

3. Graphiques de Distribution

sns.histplot(data=df, x='Colonne', bins=20)
→ Crée un histogramme avec 20 barres
sns.kdeplot(data=df, x='Colonne')
→ Crée une courbe de densité (estimateur à noyau)
sns.distplot(data, kde=True)
→ Combine histogramme et courbe de densité
sns.ecdfplot(data=df, x='Colonne')
→ Trace la fonction de répartition empirique (ECDF)
Important Puissant

4. Matrices et Corrélations

sns.heatmap(correlation_matrix, annot=True)
→ Affiche une matrice de chaleur avec valeurs (corrélations, etc.)
sns.clustermap(data)
→ Crée une heatmap avec clustering hiérarchique
corr = df.corr(); sns.heatmap(corr, annot=True, cmap='coolwarm')
→ Affiche les corrélations entre colonnes avec colormap
sns.pairplot(df)
→ Crée une matrice de graphiques pour toutes les paires de colonnes
Avancé

5. Graphiques Avancés

sns.jointplot(data=df, x='X', y='Y', kind='hex')
→ Combine 1D et 2D (scatter, kde, hist, hex, regression)
sns.regplot(data=df, x='X', y='Y')
→ Trace une courbe de régression linéaire
sns.lmplot(data=df, x='X', y='Y', hue='Categorie')
→ Régression linéaire avec plusieurs groupes (par couleur)
sns.stripplot(data=df, x='X', y='Y')
→ Affiche tous les points individuels (scatter simple)
sns.swarmplot(data=df, x='Categorie', y='Valeur')
→ Affiche tous les points en évitant les chevauchements
Important

6. Personnalisation et Sauvegarde

plt.title('Titre'); plt.xlabel('X'); plt.ylabel('Y')
→ Ajoute titre et étiquettes (fonctionne avec Seaborn)
sns.set_context('poster')
→ Définit la taille des éléments ('paper', 'notebook', 'talk', 'poster')
g = sns.FacetGrid(df, col='Categorie')
→ Crée une grille de graphiques facettisés par colonne
g.map(plt.scatter, 'X', 'Y')
→ Applique une fonction à chaque facette de la grille
plt.savefig('graphique_seaborn.png', dpi=300, bbox_inches='tight')
→ Sauvegarde avec résolution haute et ajustement automatique
Avancé Puissant

7. Syntaxes Avancées et Paramétrages Utiles

sns.scatterplot(data=df, x='X', y='Y', hue='Categorie')
→ Colore les points par catégorie (hue = teinte/couleur)
sns.scatterplot(data=df, x='X', y='Y', hue='Cat1', style='Cat2')
→ Utilise couleurs pour Cat1 et styles de marqueurs pour Cat2
sns.scatterplot(data=df, x='X', y='Y', size='Valeur')
→ Varie la taille des points selon Valeur
sns.barplot(data=df, x='X', y='Y', hue_order=['A', 'B', 'C'])
→ Définit l'ordre d'affichage des catégories
sns.scatterplot(data=df, x='X', y='Y', palette='Set2')
→ Utilise une palette nommée (Set1, Set2, husl, dark, pastel...)
sns.relplot(data=df, x='X', y='Y', hue='Cat', col='Type')
→ Crée plusieurs graphiques facettisés par colonne (figure-level)
sns.catplot(data=df, x='X', y='Y', kind='violin', hue='Cat')
→ Fonction figure-level pour graphiques catégoriques
sns.displot(data=df, x='Colonne', kde=True, hue='Categorie')
→ Figure-level pour distributions avec plusieurs groupes
sns.heatmap(corr, cmap='coolwarm', fmt='.2f', cbar=True)
→ Heatmap avec colormap personnalisée et format des nombres
sns.boxplot(data=df, x='Cat', y='Val', dodge=False)
→ dodge=False superpose les boîtes au lieu de les côte à côte
